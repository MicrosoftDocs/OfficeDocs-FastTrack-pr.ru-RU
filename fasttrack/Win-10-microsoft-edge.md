# <a name="the-new-microsoft-edge"></a>Новая версия Microsoft Edge

Преимущество FastTrack для Windows 10 предоставляет инструкции по развертыванию новой версии Microsoft Edge для клиентов Windows 10 Корпоративная, имеющих не менее 150 рабочих мест. Кроме того, если у вас есть веб-приложение или сайт, работающие с Internet Explorer и Google Chrome, и возникают проблемы совместимости, мы устраним неполадки без дополнительных затрат по [программе App Assure](Win-10-app-assure.md).

Специалист FastTrack поможет вам развернуть Microsoft Edge в Windows 10 Корпоративная с помощью Microsoft Endpoint Manager (Microsoft Endpoint Configuration Manager или Microsoft Intune).
