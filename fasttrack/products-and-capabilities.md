# <a name="products-and-capabilities"></a>Продукты и возможности

## <a name="services-and-scenarios-supported-by-fasttrack"></a>Службы и сценарии, поддерживаемые FastTrack

В этом разделе содержатся сведения о сценариях рабочих нагрузок, поддерживаемых FastTrack, и ожиданиях в исходных средах, необходимых для начала работы. В зависимости от текущей настройки мы вместе с вами создадим план исправлений, который соответствует минимальным требованиям к вашей среде для успешной встройки.

FastTrack предоставляет рекомендации, которые помогут вам в первую очередь с основными возможностями (общими для всех Microsoft Online Services), а затем с входящей каждой службой, которая может быть подходяща:

  - [Общие](#general)
  - [Безопасность и соответствие требованиям](#security-and-compliance)
  - [Office 365](#office-365)
  - [Enterprise Mobility + Security](#enterprise-mobility--security)
  - [Windows 10](#windows-10)
  - [Виртуальный рабочий стол Windows](#windows-virtual-desktop)
  - [Служба Assure для приложений](#app-assure)
  - [Microsoft Edge](#microsoft-edge)

> [!NOTE]
> Сведения о требованиях к исходной среде для Office 365 для государственных организаций США см. в статье [Требования к исходной среде для Office 365 для государственных организаций США](https://docs.microsoft.com/fasttrack/us-gov-appendix-source-environment-expectations).
